# spoof-discord
Windows için Discord bağlantı düzeltme ve dpi aracı (spoofdpi) arayüz ve otomatik kullanım desteği.

Derleme yaparken -ldflags "-H windowsgui" parametresini kullanmayı unutmayınız.

https://github.com/hdrover/discord-drover
